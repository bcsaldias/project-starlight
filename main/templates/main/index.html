Crowdsourcing Project

In machine learning, classification is the problem of identifying to which of a set of categories a new observation belongs. In some cases, solving a classification problem involves to overcome the following challenges: very few training examples, generating training instances is very costly, and training sets are far for being representative of the data.

In this project, we are developing a probabilistic model that generate low-cost questions that produce partial labels, increasing the size of the training set with less resources, keeping the accuracy levels of the final classifier. This project will only be feasible with your help, we need your talent as a labeler!

About

Each user is presented to a random series of questions. Per each answered question, the labeler is rewarded with a different amount of points depending on how accurate it has been up to the last question. The accuracy score of each labeler is updated only with certain objects (for which the research group know the ground truth).

As labeler you will have some examples and help about how to respond accurately. So, do not worry about your previous knowledge and try your best guess!
